# AI Office Suite

A comprehensive application for productivity and document management powered by AI.

## Features

- **Document Management** - Create, edit, and organize documents
- **AI Assistance** - Intelligent suggestions and content generation
- **Markdown Support** - Full markdown editing capabilities
- **Collaborative Tools** - Work together seamlessly

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the application: `npm start`
4. Open `http://localhost:3000` in your browser

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```
